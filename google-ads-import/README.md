## 📥 HOW TO IMPORT (Quick Start)

### Prerequisites
1. [ ] Google Ads account created and active
2. [ ] Billing set up (payment method on file)
3. [ ] Google Ads Editor installed (download from Google)
4. [ ] Conversion tracking code installed on landing pages (GTM recommended)
5. [ ] Stripe checkout connected to `/thank-you` page

### Import Order (Non-Negotiable)
Follow this order exactly to avoid import errors:

1. **campaigns.csv** ← Import first (creates campaign structure)
2. **ad-groups.csv** ← Then ad groups (links to campaigns)
3. **keywords.csv** ← Then keywords (links to ad groups)
4. **negative-keywords.csv** ← Then negatives (campaign-level protection)
5. **ads.csv** ← Then ads/RSAs (creative assets)
6. **extensions.csv** ← Then extensions (sitelinks, callouts, structured snippets)

### Import Steps (In Google Ads Editor)
1. File → Import → Import from CSV
2. Select CSV file from this folder
3. Verify preview: UTF-8 encoding, comma delimiter, double quote character
4. Click **Import**
5. Review imported entities in editor
6. Validate (look for red error marks)
7. Click **Post** when ready (uploads all changes to live Google Ads account)

✅ **Total import time: ~15 minutes**
✅ **Ad approval: 1–24 hours (usually 2–4 hours)**

## 🚨 CRITICAL SUCCESS FACTORS

1. **Conversion Tracking MUST fire** ✓
   - Verify with test transaction before launch
   - Without this, Google can't optimize

2. **Landing page must be mobile-optimized** ✓
   - 60–70% of traffic will be mobile
   - Page load <3 seconds
   - CTA visible above fold

3. **Urgency messaging MUST be present** ✓
   - Homepage banner: "April 6 Deadline — 11 Days Left"
   - All ads mention urgency or HMRC authority
   - Without this, close rate drops 20%+

4. **Ads must clear review within 24 hours** ✓
   - MTD/tax content is usually approved quickly
   - If ad disapproved: Read feedback, fix, resubmit

5. **Bid strategy must match urgency** ✓
   - Maximize Conversions (days 1–3)
   - Target CPA (days 6–11)
   - Bid increases on final push days
