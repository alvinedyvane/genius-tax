### Conversion Tracking Setup (Stripe Checkout)

**Goal:** Fire a Google Ads conversion event when a user completes Stripe checkout.

**Method 1 — Google Tag Manager (Recommended):**

1. Install GTM on geniustax.co.uk (add GTM snippet to `<head>` and `<body>`)
2. Create a Google Ads conversion action in Google Ads:
   - Go to: Tools → Conversions → + New Conversion
   - Type: Website
   - Name: "MTD Signup"
   - Value: Use different values for each conversion (pass plan value)
   - Count: One
3. In GTM, create a tag:
   - Tag type: Google Ads Conversion Tracking
   - Conversion ID + Label from step 2
4. Trigger: URL contains `/success` OR `stripe checkout session complete`

**Method 2 — Stripe Webhook + Server-Side (More Reliable):**
1. Create a Stripe webhook listening for `checkout.session.completed`
2. On success, redirect to `/thank-you?plan=growth&value=49`
3. Fire GTM/GA4 event on thank-you page
4. GTM picks up event and fires Google Ads conversion tag

**Pass conversion value dynamically:**
```javascript
// On thank-you page, pass the plan price
gtag('event', 'conversion', {
  'send_to': 'AW-CONVERSION_ID/CONVERSION_LABEL',
  'value': 49.00, // dynamic based on plan
  'currency': 'GBP',
  'transaction_id': 'STRIPE_SESSION_ID'
});
```

> **Critical:** Without conversion tracking, Google's algorithm can't optimise. Get this live before spending >£50.

## 6. TRACKING & MEASUREMENT

### Google Ads Conversion Tracking — Setup Steps

**Step 1: Create conversion action**
1. Google Ads → Tools & Settings → Measurement → Conversions
2. Click `+` → Website
3. Name: `MTD Signup - Stripe`
4. Category: Purchase
5. Value: Use different values → GBP
6. Count: One (one signup per click)
7. Click-through window: 30 days
8. View-through window: 1 day
9. Attribution model: Data-driven (or Last click if insufficient data)

**Step 2: Install tag**
- Copy the Global Site Tag (gtag.js) — add to `<head>` of ALL pages
- Copy the Event Snippet — add to thank-you/success page only
- Verify via Google Tag Assistant Chrome extension

**Step 3: Import GA4 goals (optional but recommended)**
- Link Google Ads account to GA4
- Import "begin_checkout" as a micro-conversion (for optimisation signal)
- Import "purchase" as primary conversion
